  **💡연산자와 피연산자** 

````
    연산자 : 연산을 수행하는 기호(+,-,*,/등);
    피연산자 : 연산자의 작업 대상(변수, 상수, 리터럴, 수식);
````
  **💡연산자의 종류**

  ````
        산술 연산자 : 숫자 데이터를 대상으로 하는 연산을 수행합니다. 주로 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/), 나머지(%) 등이 있습니다.;
        비교 연산자 : 두 값을 비교하여 관계를 평가하고 논리적인 결과를 반환합니다. 예를 들어, ==, ===, !=, !==, >, &lt;, >=, &lt;= 등이 있습니다.;
        논리 연산자 : 논리적인 조건을 평가하고 결과를 반환합니다. 주로 &&(AND), ||(OR), !(NOT) 등이 있습니다.;
        할당 연산자 : 변수에 값을 할당하는 데 사용됩니다. 예를 들어, = 연산자는 오른쪽 피연산자의 값을 왼쪽 피연산자(변수)에 할당합니다.;
        삼항 연산자 : 조건에 따라 값을 선택하여 반환합니다. 주로 condition ? expr1 : expr2 형태로 사용되며, 조건이 참이면 expr1을, 거짓이면 expr2를 반환합니다.;
        타입 연산자 : 피연산자의 데이터 타입을 평가하거나 변환하는 데 사용됩니다. 예를 들어, typeof, instanceof 등이 있습니다.;
````
