**💡연산자의 종류**

  ````
산술 연산자 : 숫자 데이터를 대상으로 하는 연산을 수행합니다. 주로 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/), 나머지(%) 등이 있습니다.;
비교 연산자 : 두 값을 비교하여 관계를 평가하고 논리적인 결과를 반환합니다. 예를 들어, ==, ===, !=, !==, >, &lt;, >=, &lt;= 등이 있습니다.;
단항 연산자 : 가장 일반적인 단항 연산자는 부호를 바꾸는 "-(음수)" 연산자와 값을 반대로 하는 "!"(논리 부정) 연산자입니다.;
할당 연산자 : 변수에 값을 할당하는 데 사용됩니다. 예를 들어, = 연산자는 오른쪽 피연산자의 값을 왼쪽 피연산자(변수)에 할당합니다.;
삼항 연산자 : 조건에 따라 값을 선택하여 반환합니다. 주로 condition ? expr1 : expr2 형태로 사용되며, 조건이 참이면 expr1을, 거짓이면 expr2를 반환합니다.;
대입 연산자 : 대입 연산자는 대개 "=" 기호로 표시되며, 왼쪽에는 변수가 오고 오른쪽에는 값을 표현하는 표현식이나 값이 옵니다. 이를 통해 변수에 값을 할당하게 됩니다.;
비트 연산자 : 비트 연산자는 주로 비트 단위의 논리 연산, 시프트(이동), 비트 단위의 AND, OR, XOR 연산 등을 수행하는 데에 활용됩니다.;
````
    
**💡연산자의 우선순위**

  ````
1. 산술 > 비교 > 논리 > 대입, 대입은 제일 마지막에 수행된다.;
2. 단항(1) > 이항(2) > 삼항(3). 단항 연산자의 우선순위가 이항 연산자보다 높다.;
3. 단항 연산자와 대입 연산자를 제외한 모든 연산의 진행방향은 왼쪽에서 오른쪽이다.;
````
